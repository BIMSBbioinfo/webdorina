Regulator-centric search

For a scenario where researchers are interested in all genes co-targeted by a set of regulators, doRiNA can be used as follows: In this example, a custom regulator file will be provided to demonstrate the workflow using a custom regulator file. Let's assume that after reading a publication by Kedde et al, we are interestd in all co-targets of PUM2 and our custom set of regulators. On the 'new search' page, we chose to upload our own bed file, and click "Go".
On the analysis page, we pick the hg19 assembly and click "Search Database". We don't want to limit the set of candidate genes, so we skip over that option. After uploading our custom regulators, "uploaded custom regulator" is already preselected. Click on the "Regulators (set A)" field, then start typing "PUM" narrow down the search to the "PUM2 PAR-CLIP" dataset. Hit return to select PUM2. You will see a "Regulators that must match" selection option show up above the regulator list. As we want co-targets, switch the match option from "any" to "all". Then click "Search doRiNA" to start the search.
